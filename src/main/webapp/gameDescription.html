Dice Game - Description

Bedienung

  • Zu Beginn des Spiels wird der Benutzer aufgefordert, einen Usernamen festzulegen, womit dieser in Folge identifiziert wird.
  • Sobald der Benutzer einen Usernamen festgelegt hat, gelangt dieser auf die "eingeloggte Seite".
  • Auf dieser Seite hat der Benutzer die Möglichkeit, eine Lobby zu erstellen oder bestehender Lobbies beizutreten.
  • Der Ersteller der Lobby wird zum Lobbyowner. Dieser kann Konfigurationen im Zusammenhang mit dem Spiel festlegen wie: Anzahl der Spieler (der Rest wird mit KIs befüllt), das Game Modus & wie viele Runden gespielt werden soll.
  • Nur der Lobbyowner kann Spielkonfigurationen vornehmen & das Spiel starten.

In den folgenden Sektionen werden die einzelnen Game Modis näher beschrieben.

In diesem ersten Würfelspiel können wir die Auswirkungen eines Systems abhängiger Ereignisse mit statistischen Schwankungen untersuchen.

  • Die Spieler werden nebeneinander angeordnet & jeder Spieler startet mit 4 Token.
  • Durch den Buttonklick "Roll Dices" kann jeder Spieler einmal pro Runde für sich selber Würfeln.
  • Der Spieler, der dran ist wird als "aktiver Spieler" angezeigt. Der aktive Spieler verschiebt für jeden Spieler durch Buttonklick "Move" die Anzahl der jeweils gewürfelten Token zum nächsten Spieler.
  • Abhängig von der gewürfelten Zahl vom ersten Spieler werden jede Runde neue Token ins Spiel eingespeist.
  • Durch den letzten Spieler werden die Token vom System wieder ausgespeist.
  • Ziel des Spiel ist es, so viele Token als möglich beim letzten Spieler aufzusammeln.

Das Prinzip beim Game Mode 2 ist derselbe wie bei Game Mode 1 nur mit dem Unterschied, dass die Würfel an einen anderen Spieler abgegeben werden können. Ziel ist es, verschiedene Strategien auszuprobieren, um am Ende des Spiels einen höheren Durchsatz zu erzielen als im Game Mode 1.

Im Game Mode 3 kann jeder Spieler für sich selbst einen Würfelbereich (entweder zwischen 1-6 oder 3-4) festlegen. Ziel ist wieder, durch die Anwendung unterschiedlicher Strategien am Ende einen höheren Durchsatz zu erreichen als im Game Mode 1.

Im Game Mode 4 sind die Würfel unsichtbar für jeden Spieler. Jeder Spieler kann pro Runde einen "weakest Link" bestimmen. Am Ende des Spiels kann man sich den in der jeweiligen Runde ausgewählten schwächsten Spieler ansehen & mit dem tatsächlich schwächsten Spieler vergleichen. Das Ziel in diesem Game Mode ist es, den schwächsten Spieler zu identifizieren.

Im Game Mode 6 kann vor Spielbeginn der Würfelbereich festgelegt werden. Das Ziel ist es, durch Verwendung unterschiedlicher Strategien einen höheren Durchsatz zu erreichen.
